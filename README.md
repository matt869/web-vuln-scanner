# SORT // VISUALIZER
A high-performance, web-based sorting algorithm visualizer built with Python and JavaScript.

## 🚀 Features
- **15 Algorithms**: Includes Tim Sort, Intro Sort, Quick Sort, and even Bogo Sort.
- **Real-world Stats**: Tracks comparisons, writes, and execution time in milliseconds.
- **Zero Dependencies**: Runs using only standard Python libraries.

## 🛠️ How to Run
1. Clone the repo: \git clone https://github.com/matt869/web-vuln-scanner.git\
2. Navigate to the folder: \cd web-vuln-scanner\
3. Run the script: \python sort_visualizer.py\
4. Open \http://127.0.0.1:8765\ in your browser.
